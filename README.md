## 🧩 Frequently Asked Questions

**Does the bot store my password?**
The software stores your credentials locally on your computer in the config.json file. It does not send these details to any outside servers. Keep this file private to protect your accounts.

**Why does the browser window pop up?**
The bot uses a real browser to interact with the Instagram website. This makes the activity look like a human user. You should leave this window open but you can place it in the background while you work on other tasks.

**What happens if the bot encounters an error?**
The terminal window will display an error message. Most errors relate to network timeouts or incorrect login details. Verify your internet and your password in the config file.

**Can I run multiple instances?**
You can run multiple instances of the program to handle different tasks. Each instance uses separate memory and browser profiles. Note that this requires more CPU power, so monitor your system performance.

**Is this safe for my accounts?**
Automation carries inherent risks. Use this tool responsibly. Avoid mass-following or overly aggressive engagement. Balance your automated tasks with manual activity to keep accounts healthy.
